# Turtlesim Joy Package

This package provides a ROS 2 node that allows control of the Turtlesim simulator using a joystick. It subscribes to joystick inputs and publishes velocity commands to the Turtlesim turtle.

To launch the package, use the following command:
```bash
ros2 launch turtlesim_joy turtlesim_joy_bringup.launch.py
```
